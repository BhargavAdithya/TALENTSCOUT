## ‚ú® Features

### AI Question Generation
- Questions generated based on candidate's tech stack and experience
- Scenario-based real-world questions
- Follow-up questions based on previous answers

### Adaptive Difficulty
- Questions adjust difficulty based on performance
- Difficulty scale: 1 (Junior) to 5 (Expert)

### Live Monitoring
- Camera monitoring during interview
- Fullscreen mode required
- Anti-cheating protection (copy/paste blocked, DevTools disabled)
- Violation tracking with 10-strike policy

### AI Evaluation
- Answers rated on technical accuracy, depth, clarity, and practical knowledge
- Final candidate rating (0-5 stars)

### Time Management
- 3-minute timer per question
- 5 questions total
- Auto-submit on timeout

### Candidate Data
- Collects name, email, phone, experience, position, locations, tech stack
- Input validation
- Duplicate prevention (3-attempt limit)
